
# 🚀 TetroBattle AI

一個充滿能量、具備 AI 評論功能的俄羅斯方塊大逃殺 Web App。

## 🌟 特色功能
- **多人連線對戰模擬**：與多位虛擬對手同時競技，爭奪排名。
- **AI 競技評論**：由 Google Gemini 驅動的高能解說，根據你的操作即時給予評價。
- **分享連結**：內建分享功能，輕鬆邀請好友加入戰場。
- **響應式設計**：支援電腦與手機瀏覽器。

## 🕹️ 遊戲操作
- **方向鍵 (←/→)**：移動方塊。
- **方向鍵 (↑)**：旋轉方塊。
- **方向鍵 (↓)**：軟掉落 (加速)。
- **空白鍵 (Space)**：硬掉落 (直接置底)。
- **C 鍵**：保留方塊 (Hold)。
- **Esc 鍵**：退出戰場。

## 📦 如何發佈此專案
1. **GitHub Pages**：
   - 將檔案上傳至 GitHub 倉庫。
   - 在 Settings > Pages 開啟自動部署。
2. **Netlify**：
   - 將專案資料夾拖放至 Netlify Drop 即可完成。
